### Tools
Zotero
- A citation manager that can also access articles.

LaTeX
- A typesetting software system that makes great-looking documents. I'm still learning to use it, but I know there's no going back to Google Docs for my papers. I wish I had started using it sooner because it would have made formatting syntax problem sets so much easier. Plus, the documents it produces are just crisp and professional, and pleasing on the eye.
